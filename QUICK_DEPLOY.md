## The 5-Step Deployment Process

### Step 1️⃣ Create Free Accounts (5 minutes)
```
✓ Vercel (vercel.com) - for frontend
✓ Render (render.com) - for backend  
✓ Neon (console.neon.tech) - for database
```

### Step 2️⃣ Set Up Database (5 minutes)
```
1. Create project on Neon
2. Copy connection string
3. Add to Render environment variables
```

### Step 3️⃣ Deploy Backend (10 minutes)
```
1. Go to Render.com
2. Click "New Web Service"
3. Select your GitHub repo
4. Add environment variables (see DEPLOYMENT_GUIDE.md)
5. Deploy! ✅
```

### Step 4️⃣ Deploy Frontend (5 minutes)
```
1. Go to Vercel.com
2. Click "New Project"
3. Select repository
4. Root dir: artifacts/classroom-assistant
5. Deploy! ✅
```

### Step 5️⃣ Test It Works (5 minutes)
```
1. Visit your Vercel URL
2. Click "Sign in with Google"
3. See your courses and events
4. Done! 🎉
```

## Need Help?

### Common Issues

**Q: "CORS error" when logging in**  
A: Update CORS in backend, redeploy

**Q: "Backend taking 30 seconds to load"**  
A: Normal on Render free tier. Upgrade to $7/mo for instant

**Q: "Can't connect to database"**  
A: Check DATABASE_URL in Render env vars, verify Neon is running

See **DEPLOYMENT_GUIDE.md** troubleshooting section for more.
